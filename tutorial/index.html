How to publish your blogpost ?

The blogpost builds upon Hugo and Markdown.

For markdown you can easily find a cheat sheet here.

  1. Setupt: you need to have a GitHub Account, a terminal and a text editor of your choice (e.g.VSCode, nano, gedit)

  2. Install Hugo on your laptop. Hugo is available for all operating systems. You can find the installation guide here.

  3. Clone the repository of the course
